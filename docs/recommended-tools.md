# Recommended skills, plugins & MCPs

A curated, opinionated list of skills, plugins, and MCP servers worth adopting for
design work with Claude — grouped by where they help. Most are external (GitHub,
mcpmarket, vendor docs); a few we also package for download in the site's
**Skills** section.

> Source of truth lives here in the repo. (Originally drafted on the PD Confluence
> page — keep this file canonical and point Confluence at it to avoid drift.)

## 🧠 General

| Tool | Use when |
| --- | --- |
| [Psychology Foundations for UX](https://mcpmarket.com/tools/skills/psychology-foundations-for-ux) | Understanding *why* UX patterns work — cognitive psychology, behavioral science, neuroscience. (Also downloadable in our [Skills section](https://design-artifacts.vercel.app/).) |
| [The "grill me" skill](https://www.aihero.dev/my-grill-me-skill-has-gone-viral) | All the time — it makes Claude interrogate your goals before acting, so better input → better output. |
| [design-workshop-prompts](https://github.com/Owl-Listener/design-workshop-prompts) | Planning a design sprint or workshop. |
| [designer-skills](https://github.com/Owl-Listener/designer-skills) | A big grab-bag — skills spanning research plans through visual critique. |

## 🎨 Design

| Tool | Use when |
| --- | --- |
| [Design notebook (idamadam)](https://www.idamadam.com/notebook) | Divergent thinking — spin up a design notebook and use the keywords *diverge / iterate / refine* to keep exploration history. (Tweak it to default to clickable prototypes instead of static — an easy fix.) |
| [design.md](https://stitch.withgoogle.com/docs/design-md/overview) — [repo](https://github.com/google-labs-code/design.md) · [video](https://www.youtube.com/watch?v=5VgnHOiOUyc) | Give Claude a source of truth for your designs (this is what our own `DESIGN.md` is built on). |
| [mood-protocol](https://github.com/Owl-Listener/mood-protocol) | Create a design moodboard for Claude to follow (complements a `design.md`). Start prototypes on the right foot and avoid default vibe-code UI. |
| [Agentation](https://www.agentation.com/) | Pointed feedback inside Claude Code — target exact elements instead of screenshotting and pasting into the terminal. |
| [bencium-marketplace](https://github.com/bencium/bencium-marketplace) | Leveling up Claude's out-of-the-box design expertise. |
| [figma-console MCP](https://docs.figma-console-mcp.southleft.com/) | A better Figma MCP — write to Figma more effectively. |
| [inclusive-design-skills](https://github.com/Owl-Listener/inclusive-design-skills) | Making designs accessible — inclusive personas, responsiveness reviews, etc. |
| [designpowers](https://github.com/Owl-Listener/designpowers) | A team of 10 design agents with different specialties — use them for prototyping in Claude Code, including running a thoughtful design critique. |

| Tool | Use when |
